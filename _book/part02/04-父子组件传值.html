04-父子组件传值

1.父组件想子组件传值

​ 父组件可以在引用子组件的时候， 通过 属性绑定（v-bind:） 的形式, 把需要传递给子组件的数据，以属性绑定的形式，传递到子组件内部，供子组件使用 

<div id="app">
    <!-- 我们把父组件中 的msg 传递到子组件，子组件通过parentmsg使用-->
    <com1 v-bind:parentmsg="msg"></com1>
</div>

定义父子组件

<script>
    var vm = new Vue({
      el: '#app',
      data: {
        msg: '123 啊-父组件中的数据'
      },
      methods: {},
      components: {
        com1: {
          data() {//子组件私有data 
            return {
              title: '123',
              content: 'qqq'
            }
          },
          template: '<h1 @click="change">这是子组件 --- {{ parentmsg }}</h1>',
          ////////////// 注意点 //////////////////
          //1.组件中的 所有 props 中的数据，都是通过 父组件传递给子组件的
          //2.props 中的数据，都是只读的，无法重新赋值
          //3.把父组件传递过来的 parentmsg 属性，先在 props 数组中定义，这样才能使用
          props: ['parentmsg'], 
          methods: {
            change() {
              this.parentmsg = '更改父组件传递过来的值'
           }
         }
       }
     }
   });
</script>

2.父组件把方法传递给子组件

<body>
  <div id="app">
    <!-- 父组件向子组件 传递 方法，使用的是 事件绑定机制； v-on, 当我们自定义了 一个 事件属性之后，那么，子组件就能够来调用传递进去的方法了 -->
    <com2 @func="show"></com2>
  </div>

  <template id="tmpl">
    <div>
      <h1>这是 子组件</h1>
      <input type="button" value="这是子组件中的按钮 - 点击它，触发 父组件传递过来的 func 方法" @click="myclick">
    </div>
  </template>

  <script>
    // 定义了一个字面量类型的 组件模板对象
    var com2 = {
      template: '#tmpl', 
      data() {
        return {
          sonmsg: { name: '小头儿子', age: 6 }
        }
      },
      methods: {
        myclick() {
          this.$emit('func', this.sonmsg)
        }
      }
    }
    var vm = new Vue({
      el: '#app',
      data: {
        datamsgFormSon: null
      },
      methods: {
        show(data) {
          console.log('调用了父组件身上的 show 方法: --- ' + data)
          this.datamsgFormSon = data
        }
      },
      components: {
        // com2: com2
        com2
      }
    });
  </script>
</body>
